========================
PRACTICAL 4A — PIPELINE SCHEDULING
========================

New Item → Name: prac4a → Select Pipeline → OK

Scroll → Build Triggers → Tick:
Build periodically

Enter CRON:
* * * * *

Scroll → Pipeline → Definition: Pipeline script

Paste:

pipeline {
 agent any
 stages {
  stage('Create File') {
   steps {
    bat 'echo Hello Jenkins > demo.txt'
   }
  }
  stage('Archive File') {
   steps {
    archiveArtifacts 'demo.txt'
   }
  }
 }
}

Click Save

Click Build Now (first manual run required)

Check:
Build → Artifacts → demo.txt

========================
PRACTICAL 4C — PARAMETERIZED PIPELINE
========================

New Item → Name: prac4c → Select Pipeline → OK

Tick:
This project is parameterized

Click Add Parameter → String Parameter

Name: USERNAME  
Default Value: Student  
Description: Enter your name

Scroll → Pipeline → Paste:

pipeline {
 agent any

 parameters {
  string(name: 'USERNAME', defaultValue: 'Student')
 }

 stages {
  stage('Greeting') {
   steps {
    echo "Hello ${params.USERNAME}"
    bat "echo Hello ${params.USERNAME} > message.txt"
   }
  }
 }
}

Click Save

Click Build with Parameters

Enter name → Run

Check:
Artifacts → message.txt
